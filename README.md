# hotel-reservation


install npm dependencies, mongoodb community server and node before running it on your local machine

In the project directory, you can run:

npm install  to install dependencies

npm start    to start the project on localhost

just paste the the below Url's to check the outputs.

End Points to check in the browser as it's just a backend task.

To create the reservation it's better to use postman


# get all the reservation

http://localhost:5000/api/reservation

# reservation get by id 

http://localhost:5000/api/reservation/id

# create reservation

http://localhost:5000/api/reservation/create

# delete reservation by id

http://localhost:5000/api/reservation/delete/id

# guest summary details

http://localhost:5000/api/reservation/summary/stay
